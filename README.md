# Retail Sales Profit Analysis
This project analyzes retail order data from 2022–2023 to understand sales trends, profitability differences, and regional performance.

The dataset contains 9,994 records and includes information about order date, region, category, price, and profit.

The main goal of this analysis is to understand sales trends, profit differences across categories, and regional performance. The project also identifies transactions that generated negative profit.

Dataset source:
https://www.kaggle.com/datasets/ankitbansal06/retail-orders�

Key findings:
Technology category has the highest total profit.

West region has the highest total sales.

3,997 transactions resulted in negative profit.

Sales increased in 2023 compared to 2022.


Tools used:
Python (Pandas, Matplotlib)
